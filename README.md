# trabalho-do-Fabiano
Funcionalidades
Cadastro de usuários com:
Nome
Email
Senha
Mensagem
Validação de campos no front-end (JavaScript) e back-end (PHP)
Armazenamento seguro de senhas
Exibição de erros no formulário

Requisitos
Servidor local com PHP (XAMPP, WAMP, MAMP ou Laragon)
MySQL ou MariaDB
Navegador moderno (Chrome, Firefox, Edge)

Como Rodar
Coloque o projeto na pasta do servidor local (htdocs no XAMPP).
Inicie Apache e MySQL.
Acesse http://localhost/index.php.
Preencha o formulário e envie.
